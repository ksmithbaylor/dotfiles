Gespeicherte Einstellungen vergessen

Beim Herstellen einer Verbindung zu einem Netzwerk können Sie das Kontrollkästchen „Einstellungen speichern“ aktivieren, damit Junos Pulse Ihre Anmeldeinformationen speichert. (Beachten Sie, dass der Netzwerkadministrator diese Funktion deaktivieren kann.) Jeder Bildschirm mit einer Eingabeaufforderung verfügt über ein eigenes Kontrollkästchen „Einstellungen speichern“. Sie werden nach dem Speichern der Einstellungen aufgefordert, Informationen zu aufeinander folgenden Anmeldeversuchen bereitzustellen. Wenn sich allerdings Ihre Anmeldeinformationen ändern, können Sie die neuen Anmeldeinformationen erst speichern, nachdem Sie die gespeicherten Einstellungen in Junos Pulse gelöscht haben. Die Funktion „Gespeicherte Einstellungen vergessen“ betrifft nicht nur die Einstellungen der ausgewählten Verbindung, sondern alle gespeicherten Einstellungen.

Wenn Sie Gespeicherte Einstellungen vergessen auswählen, während eine Verbindung unterbrochen ist, wird die Verbindung getrennt.

Die Option Verbindung unterbrechen ist nicht verfügbar, wenn Sie bei einer aktiven Verbindung Gespeicherte Einstellungen vergessen auswählen.

So löschen Sie gespeicherte Anmeldeinformationen:

  1. Klicken Sie in der Menüleiste auf Datei > Verbindungen > Gespeicherte Einstellungen vergessen.

Verwandte Themen

  • Junos Pulse-Benutzeroberfläche
  • Auf lokale Netzwerkressourcen zugreifen (Verbindung unterbrechen/fortsetzen)
