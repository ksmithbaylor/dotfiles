Junos Pulse-Verbindungsstatus

In Junos Pulse wird der Status einer Verbindung im Verbindungsbereich und in der Menüleiste angezeigt.

Ein Symbol im Verbindungsbereich zeigt den Status der einzelnen Verbindungen an. Eine Verbindung kann folgende Status aufweisen:

 Verbunden
 Verbindung wird hergestellt
 Verbunden mit Problemen
 Verbindungsherstellungsversuch fehlgeschlagen
 Verbindung unterbrochen

Ein einzelnes Menüleistensymbol gibt den Status aller aktiven Junos Pulse-Verbindungen an. Sie können auf das Menüleistensymbol klicken, um ein Menü zum Steuern der Pulse-Verbindungen anzuzeigen. Die folgende Tabelle enthält die vom Menüleistensymbol angegebenen Verbindungsstatus.

Verbunden
Keine Verbindung
Verbindung wird hergestellt
Verbunden mit Problemen
Verbindungsherstellung fehlgeschlagen
Verbindung unterbrochen
Pulse nicht ausgeführt

Verwandte Themen

  • Junos Pulse-Benutzeroberfläche
  • Junos Pulse-Taskleistensymbol verwenden und Verbindungsstatus anzeigen
