Junos Pulse-Protokolldateien

In einer Junos Pulse-Protokolldatei werden Informationen erfasst, mit denen Verbindungsprobleme gelöst werden können. Die Protokollierung erfolgt im Hintergrund. Sie müssen nur dann Änderungen an der Protokollierungsumgebung vornehmen, wenn Sie im Rahmen einer Fehlerbehebungsmaßnahme dazu aufgefordert werden.

Protokolldateien enthalten Ihre Pulse-Verbindungen und deren Eigenschaften. Um die Konfiguration zu speichern, speichern Sie alle Protokolle. Ihre aktuelle Pulse-Konfiguration wird im Protokollpaket gespeichert. Wenn Sie Pulse deinstallieren, haben Sie die Möglichkeit, Ihre Pulse-Konfiguration zu speichern, sodass Ihnen bei einer erneuten Installation Ihre Pulse-Verbindungen und andere Konfigurationseinstellungen zur Verfügung stehen.

Zur Unterstützung der Fehlerbehebung werden Sie möglicherweise aufgefordert, die folgenden Aufgaben auszuführen:

  • Protokolle mit Anmerkungen versehen – Wenn Sie einer Protokolldatei eine Anmerkung hinzufügen, fügen Sie Text als Markierung an einer bestimmten Position in der Protokolldatei ein. Zur Behebung eines Verbindungsproblems können Sie beispielsweise aufgefordert werden, spezifischen Text als Anmerkung in die Protokolldatei einzufügen, die fehlerhafte Verbindung zu testen und erneut eine Anmerkung in die Protokolldatei einzufügen. Diese Ereignisabfolge ermöglicht einem Supportmitarbeiter das Durchsuchen der Protokolldatei nach dem von Ihnen eingefügten Text. Der Text umschließt die Einträge der Protokolldatei, die das Verbindungsproblem protokollieren.
  • Protokollebene festlegen – Die standardmäßige Protokollebene ist „Normal“. Bei Fehlerhebungsvorgängen werden Sie möglicherweise aufgefordert, die Protokollebene in „Ausführlich“ zu ändern.
  • Protokolle speichern – Mit „Speichern unter“ werden alle Protokolldateien in einer einzigen Datei gesammelt. Zudem können Sie den gewünschten Speicherort angeben.

Verwandte Themen

  • Protokolldateien mit Anmerkungen versehen
  • Protokollebene festlegen
  • Protokolldateien speichern
