Informationen zur Reparatur einer Verbindung anzeigen

Wenn ein Verbindungsversuch fehlschlägt oder bei einer aktiven Verbindung ein Problem auftritt, werden von Junos Pulse Informationen zur Reparatur angezeigt. Die Informationen zur Reparatur enthalten Anweisungen zum Beheben des Problems, oder sie geben die automatisch initiierten Prozeduren zur Reparatur an.

So zeigen Sie Informationen zur Reparatur an.

Option 1

  1. Klicken Sie auf die Verbindung, um sie auszuwählen.
  2. Klicken Sie auf Datei > Verbindungen > Informationen zur Reparatur, um das Dialogfeld „Reparatur“ zu öffnen.

    Liegen keine aktiven oder ausstehenden Reparaturaktionen an, ist das Menü „Reparatur“ gedimmt.

  3. Der obere Teil des Dialogfelds beschreibt das Problem, der untere Teil die Schritte zum Beheben des Problems. Führen Sie die Schritte zur Reparatur aus, und klicken Sie auf Erneut versuchen, um die Verbindung wiederherzustellen.

Option 2

  1. Klicken Sie im Verbindungsbereich von Junos Pulse auf den Pfeil neben einer Verbindung, um die Verbindung zu erweitern und den Verbindungsstatus anzuzeigen.
  2. Klicken Sie im Verbindungsstatus auf den Link „Details“, um detailliertere Informationen zum Problem anzuzeigen.
