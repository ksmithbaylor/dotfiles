Verbindungssitzung verlängern

Der Netzwerkadministrator definiert das Sitzungszeitlimit der Netzwerkverbindungen. Der Administrator definiert das maximale Sitzungslimit und das Leerlaufzeitlimit. Sie werden von Pulse durch eine Meldung benachrichtigt, wenn Sie ein Limit erreichen. Die Meldung enthält die verbleibende Zeit in der Netzwerksitzung, bevor Pulse die Verbindung trennt. Wenn Sie keine Aktion ausführen, trennt Pulse die Verbindung, wenn der Sitzungszähler Null erreicht. Wenn Sie auf „Verlängern“ klicken, wird der Zähler auf den vom Administrator definierten Wert zurückgesetzt, und die Verbindung wird ohne Dienstunterbrechung fortgesetzt.

Sie können eine Sitzung auch verlängern, ohne darauf zu warten, dass Pulse Sie über eine ablaufende Sitzung benachrichtigt. Sie können die Sitzung jederzeit verlängern, um das Sitzungs- und das Leerlaufzeitlimit auf ihre Maximalwerte zurückzusetzen. So verlängern Sie eine Sitzung:

  1. Klicken Sie in der Pulse-Verbindungsliste auf die Verbindung, die Sie verlängern möchten, um sie auszuwählen.
  2. Klicken Sie im Pulse-Menü auf Datei > Verbindungen > Sitzung verlängern.

Oder

  1. Klicken Sie in der Menüleiste auf das Pulse-Symbol, um das Popupmenü anzuzeigen.
  2. Klicken Sie auf die aktive Verbindung und dann auf Sitzung verlängern.
