Junos Pulse-Konfigurationsübersicht

Die meisten Konfigurationsaufgaben in Junos Pulse werden vom Netzwerkadministrator ausgeführt. Der Netzwerkadministrator kann zudem Junos Pulse-Funktionen aktivieren oder deaktivieren. Ihr Administrator kann z. B. Junos Pulse so konfigurieren, dass automatisch eine Verbindung zu einem bestimmten Netzwerk hergestellt wird, oder Sie die gewünschte Verbindung auswählen lassen.

In der Regel definiert der Netzwerkadministrator die Netzwerke, auf die Sie zugreifen können. Der Netzwerkadministrator ermöglicht Ihnen gegebenenfalls die Definition zusätzlicher Verbindungen. Falls Sie einen Laptop im Büro, zuhause und unterwegs verwenden, benötigen Sie höchstwahrscheinlich eine Verbindung für den Zugriff im Büro und eine andere Verbindung für Remotestandorte. Nach dem Einrichten eines Typs einer Netzwerkverbindung kann Junos Pulse jedoch automatisch angepasst werden, damit die Verbindung bestehen bleibt. Falls Sie beispielsweise an einem Remotestandort eine Verbindung zum Unternehmensnetzwerk herstellen, den Laptop dann ohne Abmelden schließen und ins Büro zurückkehren, können Sie die Computersitzung ohne erneutes Anmelden fortsetzen. Die Portabilitätsfunktion, die als Sitzungsmigration bezeichnet wird, ist von der Konfiguration des Netzwerkzugriffs durch den Administrator abhängig.

Verwandte Themen

  • Verbindung hinzufügen
  • Verbindung mit einem Netzwerk herstellen
