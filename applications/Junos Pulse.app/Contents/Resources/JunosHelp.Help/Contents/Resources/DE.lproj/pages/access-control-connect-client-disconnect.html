Eine aktive Verbindung trennen

So trennen Sie eine Netzwerkverbindung:

Option 1

  1. Klicken Sie im Pulse-Fenster auf die Verbindung, die Sie bearbeiten möchten, um sie auszuwählen.
  2. Klicken Sie in der Menüleiste auf Datei > Verbindungen > Verbindung trennen.

Option 2

  1. Klicken Sie im Verbindungsbereich von Junos Pulse für die Verbindung, die Sie trennen möchten, auf Verbindung trennen.

Option 3

  1. Klicken Sie in der Menüleiste auf das Pulse-Symbol.
  2. Klicken Sie auf die Verbindung und dann auf Verbindung trennen.
