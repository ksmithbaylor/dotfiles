Verbindung löschen

Option 1

So löschen Sie eine Verbindung:

  1. Klicken Sie auf die Verbindung, um sie auszuwählen.
  2. Klicken Sie in der Verbindungsliste auf die Verbindung, die Sie löschen möchten, um sie auszuwählen.
  3. Klicken Sie auf die Schaltfläche „Löschen“.

  4. Sie werden aufgefordert, Ihre Entscheidung zu bestätigen, bevor die Verbindung gelöscht wird.

Option 2

  1. Klicken Sie im Pulse-Fenster auf die Verbindung, die Sie löschen möchten.
  2. Klicken Sie in der Menüleiste auf Datei > Verbindungen > Verbindung Löschen.
  3. Sie werden aufgefordert, Ihre Entscheidung zu bestätigen, bevor die Verbindung gelöscht wird.

Verwandte Themen
