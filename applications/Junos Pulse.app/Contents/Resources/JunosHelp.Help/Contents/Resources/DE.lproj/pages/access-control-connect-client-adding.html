Verbindung hinzufügen

Jede Verbindung in Junos Pulse stellt ein geschütztes Netzwerk dar. In der Regel definiert der Netzwerkadministrator die Verbindungen für Sie und deaktiviert möglicherweise die Funktion „Verbindung hinzufügen“. Falls Sie neue Verbindungen erstellen müssen, erfahren Sie die zu verwendenden Einstellungen vom Netzwerkadministrator.

So fügen Sie eine neue Verbindung hinzu:

  1. Klicken Sie im Verbindungsbereich auf die Schaltfläche Verbindung hinzufügen:

    Das Dialogfeld „Verbindung hinzufügen“ wird geöffnet.

  2. Der Netzwerkadministrator teilt Ihnen in der Regel mit, welche Option Sie für „Typ“ auswählen sollen. Wählen Sie eine der folgenden Optionen aus:
    • SSL VPN or UAC—Verwenden Sie diesen Netzwerktyp beim Herstellen einer Verbindung zu einem Junos Pulse Secure Access Service oder Junos Pulse Access Control Service.
    • SRX—Verwenden Sie diesen Netzwerktyp beim Herstellen einer Verbindung zu einem Juniper Networks SRX Series-Gerät.
  3. Geben Sie unter „Name“ einen beschreibenden Namen für diese Verbindung ein. Der angegebene Name wird im Verbindungsbereich von Pulse angezeigt.
  4. Geben Sie als Server-URL das Netzwerk an, zu dem Sie eine Verbindung herstellen möchten. Sie können die Server-URL in einem der folgenden Formate eingeben:
    • Eine IP-Adresse, zum Beispiel 10.204.71.86
    • Ein DNS-Name, zum Beispiel server.meinunternehmen.net.
  5. Klicken Sie auf Hinzufügen, um Ihre Verbindung zu speichern und das Dialogfeld zu schließen. Klicken Sie auf Verbinden, um die neue Verbindung zu speichern und eine Verbindung zum Netzwerk zu initiieren.
