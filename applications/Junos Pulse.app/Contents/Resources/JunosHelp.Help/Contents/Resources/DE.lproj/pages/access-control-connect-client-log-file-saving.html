Protokolldateien speichern

Als Teil des Fehlerbehebungsprozesses werden Sie gegebenenfalls aufgefordert, die Junos Pulse-Protokolldateien zu speichern. Pulse enthält zahlreiche mögliche Komponenten. Jede Komponente generiert mindestens eine Protokolldatei. Mit dem Vorgang „Speichern unter“ werden alle Pulse-Protokolldateien und andere Diagnosedateien in einer einzigen Datei kombiniert.

So speichern Sie Junos Pulse-Protokolldateien:

  1. Klicken Sie in der Menüleiste auf Datei > Protokolle > Speichern unter. Das Dialogfeld „Speichern unter“ wird angezeigt.
  2. Akzeptieren Sie entweder die Standardwerte für den Standort und Dateinamen, oder geben Sie neue Werte an, und klicken Sie auf Speichern.

Verwandte Themen

  • Protokolldateien mit Anmerkungen versehen
  • Protokollebene festlegen
