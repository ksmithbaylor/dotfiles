Verbindungseigenschaften bearbeiten

Nach dem Herstellen einer Verbindung können Sie die URL und den Namen bearbeiten, der im Verbindungsbereich angezeigt wird. Sie können eine Verbindung nur ändern, wenn sie derzeit nicht aktiv ist.

So bearbeiten Sie eine Verbindung:

Option 1

  1. Klicken Sie im Pulse-Fenster auf die Verbindung, die Sie bearbeiten möchten, um sie auszuwählen.
  2. Klicken Sie in der Menüleiste auf Datei > Verbindungen > Verbindung bearbeiten.

    Das Dialogfeld „Verbindung bearbeiten“ wird geöffnet.

  3. Bearbeiten Sie die Verbindung, und klicken Sie dann auf Speichern, um die Änderungen zu speichern und das Dialogfeld zu schließen, oder klicken Sie auf Verbinden, um eine Verbindung herzustellen und das Dialogfeld zu schließen.

Option 2

  1. Klicken Sie auf die Verbindung, um sie auszuwählen.
  2. Klicken Sie auf die Schaltfläche „Verbindung bearbeiten“:

    Das Dialogfeld „Verbindung bearbeiten“ wird geöffnet.

  3. Bearbeiten Sie die Verbindung, und klicken Sie dann auf Speichern, um die Änderungen zu speichern und das Dialogfeld zu schließen, oder klicken Sie auf Verbinden, um eine Verbindung herzustellen und das Dialogfeld zu schließen.

Verwandte Themen

  • Verbindungseigenschaften anzeigen
  • Informationen zur Reparatur einer Verbindung anzeigen
