Junos Pulse-Menüleistensymbol verwenden und Verbindungsstatus anzeigen

Das Junos Pulse-Menüleistensymbol gibt den aktuellen Status der Netzwerkverbindung an. Sie sehen ein Pulse-Symbol in der Menüleiste, selbst wenn derzeit mehr als eine Verbindung besteht. Bei einem Problem mit einer aktiven Verbindung ändert das Symbol seinen Status. In der folgenden Liste werden die Status des Menüleistensymbols beschrieben.

Verbunden
Keine Verbindung
Verbindung wird hergestellt
Verbunden mit Problemen
Verbindungsherstellung fehlgeschlagen
Verbindung unterbrochen
Pulse nicht ausgeführt

Sie können auch auf das Menüleistensymbol klicken, um ein Menü anzuzeigen und die folgenden Aktionen auszuführen:

  • Junos Pulse öffnen – Zeigt die Pulse-Benutzeroberfläche an. Wenn Pulse nicht ausgeführt wird, wird beim Öffnen von Junos Pulse zunächst der Pulse-Dienst gestartet.
  • Verbindungsliste – Das Menü führt jede Pulse-Verbindung auf. Klicken Sie auf den Verbindungsnamen, um die Aktionen anzuzeigen, die Sie auf dieser Verbindung ausführen können:

    • Verbindung herstellen – Eine Verbindung zu dem ausgewählten Netzwerk herstellen.
    • Trennen – Beendet die ausgewählte Netzwerkverbindung.
    • Abbrechen – Eine Aktion abbrechen, die von einem Pulse-Server eingeleitet wurde. Die Option „Abbrechen“ steht zur Verfügung, wenn der Server einen Host Checker-Vorgang eingeleitet hat. Wenn Sie abbrechen, stellt Pulse keine Verbindung zu dem Netzwerk her.
    • Unterbrechen/Fortsetzen – Unterbricht eine Verbindung zu einem Remotenetzwerk, um den Zugriff auf lokale Netzwerkressourcen wie beispielsweise einen lokalen Drucker zu ermöglichen, und setzt die Verbindung anschließend wieder fort. Wenn Sie eine Verbindung unterbrechen, verlieren Sie solange den Zugang zu geschützten Netzressourcen, bis Sie die Verbindung wieder fortsetzen. Wenn Sie eine Verbindung fortsetzen, wird die Verbindung wiederhergestellt, ohne dass Sie zur Eingabe Ihrer Anmeldeinformationen aufgefordert werden.
    • Sitzung verlängern – Sie können eine aktive Verbindungssitzung verlängern, ohne darauf zu warten, dass Pulse Sie über eine ablaufende Sitzung benachrichtigt. Sie können diese Option jederzeit auswählen, um das Sitzungs- und das Leerlaufzeitlimit auf ihre Maximalwerte zurückzusetzen.
  • Beenden – Schließt alle aktiven Verbindungen und beendet Junos Pulse. Das Pulse-Menüleistensymbol wird weiterhin angezeigt, nachdem Sie Junos Pulse beendet haben, und Sie können es verwenden, um Pulse erneut zu starten.

Verwandte Themen

  • Junos Pulse-Benutzeroberfläche
  • Junos Pulse starten oder beenden
