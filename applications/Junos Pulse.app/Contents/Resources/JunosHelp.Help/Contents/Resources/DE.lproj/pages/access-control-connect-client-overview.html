Junos Pulse-Übersicht

Junos Pulse ermöglicht sichere, authentifizierte Verbindungen zu geschützten Netzwerkressourcen über LANs und WANs. Junos Pulse vereinfacht Netzwerkverbindungen, da die Netzwerkzugriffsumgebung durch den Netzwerkadministrator konfiguriert wird.

Mit einer sicheren Verbindung wird der Netzwerkverkehr verschlüsselt, damit nur das Endgerät und der Server, mit denen kommuniziert wird, den Inhalt entschlüsseln können. Ihr Netzwerkadministrator definiert die Sicherheitseinstellungen, die von Junos Pulse beim Herstellen einer Netzwerkverbindung verwendet werden. Der Netzwerkadministrator kann Junos Pulse zudem so konfigurieren, dass für sichere Kommunikation erforderliche Software automatisch installiert wird.

Eine authentifizierte Verbindung bedeutet, dass das Endgerät und der Server ihre Identität gegenseitig nachgewiesen haben. Der Server verwendet ein Zertifikat von einer vertrauenswürdigen Zertifizierungsstelle zur eigenen Authentifizierung. Das Endgerät unterbreitet dem Server Ihre Anmeldeinformationen (ein Benutzername und ein Kennwort). Ihr Netzwerkadministrator kann Ihre Umgebung so konfigurieren, dass die Authentifizierung automatisch erfolgt, ohne dass Sie zur Eingabe von Informationen aufgefordert werden, auch wenn Sie in der Regel beim Herstellen der Verbindung nach Benutzername und Kennwort gefragt werden.

Bei einer geschützten Netzwerkressource handelt es sich um ein beliebiges Netzwerkgerät oder eine Anwendung, auf das/die Sie über eine authentifizierte Verbindung zugreifen. Zum Zugreifen auf das E-Mail-System des Unternehmens melden Sie sich beispielsweise erst am Netzwerk an. Der Netzwerkadministrator organisiert geschützte Netzwerkressourcen in Bereichen und autorisiert Ihren Zugriff auf diese Ressourcen, indem er Zugriff auf ausgewählte Bereiche gewährt. Falls Sie Zugriff auf mehrere Bereiche haben, können Sie den gewünschten Bereich auswählen, wenn Sie mit Junos Pulse eine Verbindung zum Netzwerk herstellen. Der Netzwerkadministrator kann außerdem die Sicherheitseigenschaften definieren, die für den Zugriff auf einen Bereich erforderlich sind, indem er das Host Checker-Programm aktiviert. Host Checker wird ausgeführt, wenn Sie über Junos Pulse eine Verbindung zum Netzwerk herstellen. Er prüft Ihr Endgerät, um zu gewährleisten, dass es über die erforderliche Software verfügt.

Der Netzwerkadministrator kann Zugriffsrichtlinien konfigurieren, sodass die Verbindung nach dem Herstellen der Netzwerkverbindung portabel ist. Falls Sie z. B. mit dem Laptop zuhause eine Verbindung zum Unternehmensnetzwerk herstellen, müssen Sie diese nicht trennen und eine neue Verbindung herstellen, wenn Sie den Laptop wieder im Büro verwenden. Sie schließen den Laptop einfach zuhause. Im Büro öffnen Sie ihn wieder. Junos Pulse erkennt die veränderte Netzwerkumgebung. Junos Pulse verwendet die Verbindungsprofilinformationen zur Konfiguration der Verbindung für den entsprechenden Zugriff und authentifiziert Sie in der neuen Umgebung. Falls der Administrator die Standorterkennungsfunktion von Junos Pulse konfiguriert hat, werden Sie beim Start mit dem entsprechenden Netzwerk verbunden. Falls z. B. das Endgerät an einem Remotestandort gestartet wird, wird von Junos Pulse automatisch eine Verbindung zu einem WAN-Zugriffsgerät hergestellt.
