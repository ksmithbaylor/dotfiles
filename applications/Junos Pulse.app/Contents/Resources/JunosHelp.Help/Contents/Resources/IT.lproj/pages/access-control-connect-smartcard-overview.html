Autenticazione smart card

Per l'autenticazione con smart card, si utilizza una scheda di plastica contenente le informazioni di autenticazione dell'utente. Se l'ambiente è configurato con più certificati digitali, Pulse richiede all'utente di selezionarne uno. In genere, se nell'endpoint è configurato un lettore di smart card, Pulse interagisce con il software della smart card per recuperare il codice token dopo l'immissione del codice PIN da parte dell'utente.
