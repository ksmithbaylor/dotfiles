File di registro di Junos Pulse

Un file di registro di Junos Pulse tiene traccia delle informazioni che possono agevolare la risoluzione dei problemi di connessione. L'operazione di creazione dei registri avviene in background. Non è necessario apportare alcuna modifica nell'ambiente di registrazione, a meno che non vengano fornite istruzioni in tal senso nell'ambito di un'attività di risoluzione dei problemi.

I file di registro includono le connessioni Pulse e le relative proprietà. Se si desidera salvare la configurazione, è sufficiente salvare tutti i registri. La configurazione corrente di Pulse viene salvata nel pacchetto dei registri. Se si disinstalla Pulse, il programma richiede se salvare la configurazione di Pulse in modo che le connessioni e le altre impostazioni di configurazione di Pulse siano comunque disponibili quando si installa nuovamente Pulse.

Per agevolare la risoluzione dei problemi, potrebbe essere richiesto di eseguire le seguenti attività:

  • Annotare i registri: quando si annota un file di registro, si inserisce un testo che contrassegna il file di registro in una posizione specifica. Ad esempio, per risolvere un problema di connessione, potrebbe essere richiesto di annotare il file di registro con un testo specifico, tentare la connessione che non riusciva e quindi annotare nuovamente il file di registro. Questa sequenza di eventi consente a un rappresentante dell'assistenza di ricercare nel file di registro il testo inserito. Il testo racchiude tra parentesi le voci del file di registro che tengono traccia del problema di connessione.
  • Impostare il livello di registrazione: il livello di registrazione predefinito è Normal (Normale). Per un'operazione di risoluzione dei problemi, potrebbe essere richiesto di cambiare il livello di registrazione in Detailed (Dettagliato).
  • Salvare i registri: l'operazione di salvataggio con nome raccoglie tutti i file di registro in un singolo file .zip e consente di specificare il percorso in cui posizionare il file.
