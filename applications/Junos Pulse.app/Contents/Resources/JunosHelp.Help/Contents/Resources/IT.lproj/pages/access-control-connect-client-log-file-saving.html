Salvare i file di registro

Nell'ambito di una processo di risoluzione di un problema, è possibile che venga richiesto di salvare i file di registro di Junos Pulse. Pulse include vari possibili componenti e ognuno di essi genera uno o più file di registro. L'operazione di salvataggio con nome riunisce tutti i file di registro di Pulse e altri file diagnostici in un singolo file.

Per salvare i file di registro di Junos Pulse:

  1. Nella barra dei menu, fare clic su File > Logs > Save as (File > Registri > Salva con nome). Viene visualizzata la finestra di dialogo Save As (Salva con nome).
  2. Accettare i valori predefiniti relativi al percorso e al nome file o specificare nuovi valori, quindi fare clic su Save (Salva).
