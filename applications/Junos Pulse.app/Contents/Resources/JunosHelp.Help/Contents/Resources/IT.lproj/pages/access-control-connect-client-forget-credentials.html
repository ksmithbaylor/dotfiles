Non ricordare le impostazioni salvate

Quando si esegue la connessione a una rete, è possibile selezionare la casella di controllo Save Settings (Salva impostazioni) in modo tale che Junos Pulse memorizzi le credenziali di accesso (notare che l'amministratore di rete può disabilitare questa funzione). Ogni diversa schermata in cui viene richiesta l'immissione di una risposta dispone di una propria casella di controllo Save Settings (Salva impostazioni). Quando si salvano impostazioni, ai successivi tentativi di accesso non verrà più richiesto di fornire le corrispondenti informazioni. Se le credenziali di accesso cambiano, è tuttavia necessario cancellare le impostazioni salvate in Junos Pulse prima di salvare le nuove credenziali. L'opzione Forget Saved Settings (Non ricordare le impostazioni salvate) influisce su tutte le impostazioni salvate, non solo su quelle della connessione selezionata.

Se si seleziona Forget Saved Settings (Non ricordare le impostazioni salvate) mentre una connessione è in stato di sospensione, la connessione viene interrotta.

L'opzione Sospendi non è disponibile se si sceglie Forget Saved Settings (Non ricordare le impostazioni salvate) mentre una connessione è attiva.

Per cancellare le credenziali di accesso salvate:

  1. Nella barra dei menu, fare clic su File > Connections > Forget Saved Settings (File > Connessioni > Non ricordare le impostazioni salvate).
