Annotare i file di registro

Quando si annota un file di registro, si inserisce un testo che contrassegna il file di registro in una posizione specifica. Ad esempio, per risolvere un problema di connessione, potrebbe essere richiesto di annotare il file di registro con un testo specifico, tentare la connessione che non riusciva e quindi annotare nuovamente il file di registro. Questa sequenza di eventi consente a un tecnico dell'assistenza di individuare facilmente le voci del file di registro che tengono traccia del problema di connessione.

Per annotare i file di registro di Junos Pulse:

  1. Nella barra dei menu, fare clic su File > Logs > Annotate (File > Registri > Annota) per aprire la finestra di dialogo Annotate Logs (Annota registri).
  2. Immettere il testo di annotazione, quindi fare clic su OK.

Argomenti correlati

  • Impostare il livello di registrazione
  • Salvare i file di registro
