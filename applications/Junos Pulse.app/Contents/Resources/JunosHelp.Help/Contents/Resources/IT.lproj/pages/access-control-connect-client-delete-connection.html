Eliminare una connessione

Opzione 1

Per eliminare una connessione:

  1. Fare clic sulla connessione per selezionarla.
  2. Nell'elenco Connections (Connessioni) di Pulse, fare clic sulla connessione che si desidera eliminare per selezionarla.
  3. Fare clic sul pulsante Delete (Elimina):

  4. Prima di eliminare la connessione, viene richiesto di confermare la scelta.

Opzione 2

  1. Nella finestra di Pulse, fare clic sulla connessione che si desidera eliminare.
  2. Nella barra dei menu, fare clic su File > Connections > Delete (File > Connessioni > Elimina).
  3. Prima di eliminare la connessione, viene richiesto di confermare la scelta.
