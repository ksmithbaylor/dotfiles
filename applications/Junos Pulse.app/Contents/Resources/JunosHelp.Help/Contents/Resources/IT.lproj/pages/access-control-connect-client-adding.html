Aggiungere una connessione

Ogni connessione visualizzata in Junos Pulse rappresenta una rete protetta. In genere, è l'amministratore di rete che definisce le connessioni e potrebbe disabilitare la funzione Add a Connection (Aggiungi una connessione). Se occorre creare nuove connessioni, l'amministratore di rete indicherà le impostazioni da utilizzare.

Per aggiungere una nuova connessione:

  1. Nel pannello Connections (Connessioni), fare clic sul pulsante Add a Connection (Aggiungi una connessione):

    Viene visualizzata la finestra di dialogo Add Connection (Aggiungi connessione).

  2. In Type (Tipo), l'opzione da selezionare sarà in genere indicata dall'amministratore di rete. Scegliere una delle seguenti opzioni:
    • SSL VPN or UAC (SSL VPN o UAC) — Utilizzare questo tipo di rete per le connessioni a Junos Pulse Secure Access Service o a Junos Pulse Access Control Service.
    • SRX—Utilizzare questo tipo di rete per la connessione a un dispositivo della serie Juniper Networks SRX.
  3. In Name (Nome), specificare un nome descrittivo per la connessione. Il nome specificato viene visualizzato nel pannello Connections (Connessioni) di Pulse.
  4. In Server URL (URL server), specificare la rete a cui connettersi. Tale URL può essere immesso in uno dei seguenti formati:
    • Indirizzo IP, ad esempio 10.204.71.86
    • Nome DNS, ad esempio server.società.net.
  5. Fare clic su Add (Aggiungi) per salvare la nuova connessione e chiudere la finestra di dialogo. Fare clic su Connect (Connetti) per salvare la nuova connessione e avviare una connessione alla rete.
