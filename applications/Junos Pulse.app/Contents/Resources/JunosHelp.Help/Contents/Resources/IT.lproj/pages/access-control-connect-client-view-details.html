Visualizzare i dettagli avanzati di una connessione

Il pannello Connections (Connessioni) mostra le informazioni di base relative a una connessione. È inoltre possibile visualizzare i dettagli avanzati di una connessione selezionata per informazioni più approfondite.

Per visualizzare i dettagli avanzati di una connessione:

Opzione 1

  1. Fare clic sulla connessione per selezionarla.
  2. Nella barra dei menu, fare clic su File > Connections > Advanced Connection Details (File > Connessioni > Dettagli avanzati connessione).

Opzione 2

  1. Fare clic con il pulsante destro del mouse sulla connessione Pulse per visualizzare il menu a comparsa.
  2. Scegliere Advanced Connection Details (Dettagli avanzati connessione).
