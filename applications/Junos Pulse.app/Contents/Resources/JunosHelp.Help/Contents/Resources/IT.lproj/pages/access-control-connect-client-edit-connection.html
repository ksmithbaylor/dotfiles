Modificare le proprietà di una connessione

Dopo aver creato una connessione, è possibile modificarne l'URL e il nome visualizzato nel pannello Connections (Connessioni). Una connessione può essere modificata solo se non è attualmente attiva.

Per modificare una connessione:

Opzione 1

  1. Nella finestra di Pulse, fare clic sulla connessione che si desidera modificare per selezionarla.
  2. Nella barra dei menu, fare clic su File > Connections > Edit (File > Connessioni > Modifica).

    Viene visualizzata la finestra di dialogo Edit Connection (Modifica connessione).

  3. Modificare la connessione, quindi fare clic su Save (Salva) per salvare le modifiche e chiudere la finestra di dialogo o fare clic su Connect (Connetti) per avviare una connessione e chiudere la finestra di dialogo.

Opzione 2

  1. Fare clic sulla connessione per selezionarla.
  2. Fare clic sul pulsante Edit Connection (Modifica connessione):

    Viene visualizzata la finestra di dialogo Edit Connection (Modifica connessione).

  3. Modificare la connessione, quindi fare clic su Save (Salva) per salvare le modifiche e chiudere la finestra di dialogo o fare clic su Connect (Connetti) per avviare una connessione e chiudere la finestra di dialogo.
