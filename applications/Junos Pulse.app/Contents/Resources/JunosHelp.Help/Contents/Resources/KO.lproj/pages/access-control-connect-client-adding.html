연결 추가

Junos Pulse에서 각 연결은 보호된 네트워크를 나타냅니다. 일반적으로 네트워크 관리자는 사용자를 위해 연결을 정의하고 연결 추가 기능을 비활성화할 수 있습니다. 사용자가 새로운 연결을 만들어야 한다면, 어떤 설정을 사용해야 하는지 네트워크 관리자가 알려줍니다.

새 연결 추가 방법:

  1. 연결 창에서 연결 추가 버튼을 클릭합니다.

    연결 추가 대화 상자가 열립니다.

  2. 유형의 경우, 일반적으로 네트워크 관리자가 선택할 옵션에 대해 알려줍니다. 다음 중 하나를 선택합니다.
    • SSL VPN 또는 UAC—Junos Pulse Secure Access Service 또는 Junos Pulse Access Control Service에 연결하는 경우 이 네트워크 유형을 사용합니다.
    • SRX—Juniper Networks SRX 시리즈 서비스 장치에 연결하는 경우 이 네트워크 유형을 사용합니다.
  3. 이름의 경우, 이 연결에 대한 설명이 포함된 이름을 지정하십시오. 지정한 이름은 Pulse 연결 창에 표시됩니다.
  4. 서버 URL의 경우, 연결하려는 네트워크를 지정하십시오. 다음 형식으로 서버 URL을 입력할 수 있습니다.
    • IP 주소(예: 10.204.71.86)
    • DNS 이름(예: server.mycompany.net)
  5. 추가를 클릭하여 새로운 연결을 저장하고 대화 상자를 닫습니다. 연결을 클릭하여 새로운 연결을 저장하고 네트워크로의 연결을 시작합니다.
