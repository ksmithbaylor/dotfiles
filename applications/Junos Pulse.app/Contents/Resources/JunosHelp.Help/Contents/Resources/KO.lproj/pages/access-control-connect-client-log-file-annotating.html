로그 파일에 주석 달기

로그 파일에 주석을 다는 경우 특정 위치에 로그 파일을 표시하는 텍스트를 삽입합니다. 예를 들어, 연결 문제를 해결하기 위해서 특정 텍스트로 로그 파일에 주석을 단 후, 실패한 연결을 시도하고 다시 로그 파일에 주석을 달도록 요청받을 수 있습니다. 이 같은 일련의 이벤트로 지원 담당자는 사용자의 연결 문제를 추적하는 로그 파일 항목을 쉽게 찾을 수 있게 됩니다.

Junos Pulse 로그 파일에 주석을 다는 방법:

  1. 메뉴 모음에서 파일 > 로그 > 주석 달기를 클릭하여 로그에 주석 달기 대화 상자를 엽니다.
  2. 주석 텍스트를 입력한 다음 확인을 클릭합니다.

관련 항목

  • 로그 수준 설정
  • 로그 파일 저장
