Junos Pulse 메뉴 모음 아이콘 사용 및 연결 상태 보기

Junos Pulse 메뉴 모음 아이콘은 사용자의 현재 네트워크 연결 상태를 나타냅니다. 사용자가 현재 둘 이상의 연결을 가지고 있다고 해도 메뉴 모음에는 하나의 Pulse 아이콘만 표시됩니다. 활성 상태의 연결에 문제가 생긴 경우, 아이콘의 상태가 변경됩니다. 다음 목록에는 메뉴 모음 아이콘 상태가 나열되어 있습니다.

또한 메뉴 모음 아이콘을 클릭하여 메뉴를 표시하고 다음 작업을 수행할 수 있습니다.

  • Junos Pulse 열기 - Pulse 인터페이스를 표시합니다. Pulse가 실행되고 있지 않은 경우 Junos Pulse 열기에서 먼저 Pulse 서비스를 시작합니다.
  • 연결 목록 - 메뉴에 각 Pulse 연결이 나열됩니다. 연결에 대해 수행할 수 있는 작업을 표시하려면 해당 연결 이름을 클릭합니다.

    • 연결 - 선택한 네트워크에 대한 연결을 설정합니다.
    • 연결 해제 - 선택한 네트워크의 연결을 종료합니다.
    • 취소 - Pulse 서버에서 시작한 작업을 취소합니다. 취소 옵션은 서버에서 호스트 체커 작업을 시작한 경우에 사용할 수 있습니다. 취소하면 Pulse가 네트워크에 연결되지 않습니다.
    • 일시 중단/다시 시작 - 로컬 프린터와 같은 로컬 네트워크 리소스에 액세스하려면 원격 네트워크에 대한 연결을 일시 중단한 다음 다시 시작합니다. 연결을 일시 중단하면 다시 시작할 때까지 보호된 네트워크 리소스에 대한 액세스가 끊어집니다. 연결을 다시 시작하면 로그인 자격 증명을 입력하도록 요청하는 메시지가 표시되지 않고 바로 연결됩니다.
    • 세션 연장 - 사용자는 Pulse에서 세션 만료를 통지할 때까지 기다리지 않고 활성 연결 세션을 연장할 수 있습니다. 세션 및 유휴 타이머를 최대값으로 재설정하려면 언제든지 이 옵션을 선택합니다.
  • 종료 - 활성 상태의 연결을 모두 닫고 Pulse를 중단합니다. Pulse를 종료한 후에도 Pulse 메뉴 모음 아이콘이 메뉴 모음에 남아 있어 다시 시작할 때 사용할 수 있습니다.
