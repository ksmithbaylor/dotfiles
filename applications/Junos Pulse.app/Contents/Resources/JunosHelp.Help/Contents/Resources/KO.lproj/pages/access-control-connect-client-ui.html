Junos Pulse 인터페이스

Junos Pulse 인터페이스는 네트워크 연결을 표시하고 종점의 연결 상태에 대한 정보를 제공합니다.

Junos Pulse 연결 창은 연결 작업 버튼을 표시하고 사용자의 네트워크 연결을 목록화하고 사용자가 연결 및 연결 해제를 할 수 있게 합니다. 연결 표시를 확대하고 자세한 상태를 보려면 연결 이름 옆에 있는 화살표를 클릭합니다. 일반적으로 네트워크 관리자가 사용자의 연결을 정의하게 되지만, 네트워크 관리자는 사용자가 자신의 연결을 정의하도록 할 수도 있습니다.

연결 창 버튼:

 새 연결 추가
 선택한 비활성 연결 편집
 선택한 비활성 연결 삭제

Junos Pulse는 또한 메뉴 모음 아이콘을 표시하여 연결 상태를 알려 주고, 연결 및 연결 해제 작업에 빠르게 액세스할 수 있도록 해줍니다. 하나의 메뉴 모음 아이콘으로 모든 활성 연결에 대한 상태가 제공됩니다.
