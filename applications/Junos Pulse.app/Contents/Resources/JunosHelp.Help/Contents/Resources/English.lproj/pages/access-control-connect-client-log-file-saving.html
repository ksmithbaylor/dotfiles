Save Log Files

As part of a troubleshooting process, you might be asked to save the Junos Pulse log files. Pulse includes a number of possible components and each component generates one or more log files. The Save As operation combines all of the Pulse log files and other diagnostic files into a single file.

To save Junos Pulse log files:

  1. On the menu bar, click File > Logs > Save As. The Save As dialog box appears.
  2. Either accept the default values for location and filename or specify new values, and then click Save.
