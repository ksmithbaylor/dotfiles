Set Log Level

To set the log level:

  1. On the menu bar, click File > Logs > Log Level > Detailed or File > Logs > Log Level > Normal. A check mark indicates the option that is currently enabled.

    2. Normal logging is the default. You should set logging to Normal unless you are troubleshooting a connection issue.

    Detailed logging creates a greater number of log entries and increases the size of the log files. Detailed logging is typically enabled only when troubleshooting an issue.
