Use the Junos Pulse Menu Bar Icon and View Connection Status

The Junos Pulse menu bar icon indicates the current state of your network connection. You see one Pulse icon in the menu bar even when you have more than one current connection. When any active connection has an issue, the icon changes its state. The following list describes the menu bar icon states.

Connected
No connection
Connecting
Connected with issues
Connection failed
Connection suspended
Pulse is not running

You can also click the menu bar icon to display a menu and perform the following actions:

  • Open Junos Pulse - Display the Pulse interface. If Pulse is not running, Open Junos Pulse first starts the Pulse service.
  • Connection list - The menu lists each Pulse connection. Click the connection name to display the actions you can perform on that connection:

    • Connect - Establish a connection to the selected network.
    • Disconnect - End the selected network connection.
    • Cancel - Cancel an action that was initiated by the Pulse server. The Cancel option is available when the server has initiated a Host Checker operation. If you cancel, Pulse does not connect to the network.
    • Suspend/Resume - Suspend and then resume a connection to a remote network to enable access to local network resources such as a local printer. When you suspend a connection, you lose access to protected netwrok resources until you resume the connection. When you resume a connection, you are connected without being prompted for login credentials.
    • Extend Session - You can extend an active connection session without waiting for Pulse to notify you of an expiring session. Select this option at any time to reset the session and idle timers to their maximum values.
  • Exit - Close any active connections and stop Pulse. The Pulse menu bar icon remains on the menu bar after you exit from Pulse and you can use it to start Pulse again.
