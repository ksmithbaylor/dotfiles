Add a Connection

Each connection in Junos Pulse represents a protected network. Typically, your network administrator defines the connections for you and might disable the Add a Connection feature. If you are required to create new connections, your network administrator will tell you the settings that you must use.

To add a new connection:

  1. On the Connections pane, click the Add a Connection button:

    The Add Connection dialog box opens.

  2. For Type, typically your network administrator will tell you which option to select. Choose one of the following:
    • SSL VPN or UAC—Use this network type if you are connecting to Junos Pulse Secure Access Service or Junos Pulse Access Control Service.
    • SRX—Use this network type if you are connecting to a Juniper Networks SRX Series device.
  3. For Name, specify a descriptive name for this connection. The name you specify appears in the Pulse Connections pane.
  4. For Server URL, specify the network that you want to connect to. You can enter the Server URL in any of the following formats:
    • An IP address, for example 10.204.71.86
    • A DNS name, for example server.mycompany.net.
  5. Click Add to save your new connection and close the dialog box. Click Connect to save your new connection and initiate a connection to the network.
