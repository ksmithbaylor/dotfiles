Junos Pulse Connection Status

Junos Pulse displays the status of a connection in the Connections pane and in the menu bar.

A Connections pane icon shows the state of each connection. A connection can be in any of the following states:

 Connected
 Connecting
 Connected with issues
 Connection attempt failed
 Connection suspended

A single menu bar icon indicates the status of all active Junos Pulse connections. You can click the menu bar icon to display a menu for controlling Pulse connections. The following table shows the connection status indicated by the menu bar icon.

Connected
No connection
Connecting
Connected with issues
Connection failed
Connection suspended
Pulse is not running
