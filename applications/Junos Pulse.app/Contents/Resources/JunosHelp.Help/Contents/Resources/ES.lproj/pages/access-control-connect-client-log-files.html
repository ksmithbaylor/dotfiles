Archivos de registro de Junos Pulse

Un archivo de registro de Junos Pulse rastrea la información que puede ayudarle a resolver problemas de conexión. El registro es una operación que se ejecuta en segundo plano. No es necesario que haya ningún cambio en su entorno de registro a menos que se le indique que lo haga como parte de un intento de solución de problemas.

Los archivos de registro incluyen conexiones de Pulse y sus propiedades. Si desea guardar la configuración, solo tiene que guardar todos los registros. Su configuración actual de Pulse se guarda en el paquete de registro. Si desinstala Pulse, el programa le pregunta si desea guardar su configuración de Pulse para que, cuando vuelva a instalarlo, estén disponibles sus conexiones de Pulse y otros ajustes de configuración.

Para ayudar en la solución de problemas, puede que se le solicite que realice las siguientes tareas:

  • Anotar los registros: cuando se anota un archivo de registro, se inserta texto que marca el archivo de registro en una ubicación específica. Por ejemplo, para solucionar un problema de conexión, puede que le soliciten anotar el archivo de registro con texto específico, intentar establecer la conexión que ha estado fallando y luego anotar el archivo de registro de nuevo. Esta secuencia de eventos permite que un representante de asistencia busque el archivo de registro para ver el texto que insertó. El texto incluye entre corchetes las entradas del archivo de registro que rastrean el problema de conexión.
  • Establecer nivel de registro: el nivel de registro predeterminado es Normal. Para una operación de solución de problemas, se le puede pedir que cambie el nivel de registro a Detallado.
  • Guardar los registros: la operación Guardar como reúne todos los archivos de registro en un archivo y le permite especificar dónde desea guardar el archivo.
