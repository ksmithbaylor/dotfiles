Autenticación con token

Pulse admite el sistema de autenticación con token RSA SecurID. La autenticación con token, también llamada autenticación de dos factores, es un método de generación de credenciales de autenticación que son muy difíciles de comprometer. Un código token es un número que se genera simultáneamente tanto en el cliente como en el servidor. El usuario crea un número de identificación personal (PIN). A continuación, el nombre de usuario y el PIN más el código de token actual permiten al usuario iniciar sesión en el servidor. Un PIN más un código token forman una contraseña. Un código token caduca después de un breve intervalo de tiempo, por lo general un minuto. A continuación, se reemplaza por un nuevo código token.

La primera vez que utilice el sistema, Pulse le pedirá que cree un PIN. Si el administrador ya le ha proporcionado un PIN, se le solicitará que lo cambie. Si el administrador de red requiere que cambie el PIN periódicamente, Pulse va a solicitarle un PIN nuevo. Por lo general, debe proporcionar el código token actual al cambiar el PIN. En algunos casos, después de introducir la contraseña, se le solicitará que especifique el siguiente código token antes de acceder a la red. Espere hasta que su dispositivo de token cambie a un nuevo código token y a continuación introduzca ese número para continuar.

Autenticadores

El dispositivo o software que genera el token se denomina autenticador. Pulse admite la gama completa de autenticadores de hardware SecurID.

Los autenticadores de hardware (físicos) de RSA SecurID muestran un nuevo código token cada 60 segundos. Cuando vaya a iniciar sesión, deberá tener con usted la tarjeta o el llavero que genera el token. En la ventana de inicio de sesión de Pulse, introduzca su nombre de usuario y su PIN más el código token actual. Un autenticador USB permite almacenar sus credenciales de contraseña y nombre de usuario de MacOS, así como sus certificados digitales. Cuando el dispositivo está conectado a su equipo, este permite la introducción automática del código token y Pulse puede obtener acceso de manera automática a los códigos token directamente desde el dispositivo, una vez que haya introducido su PIN.
