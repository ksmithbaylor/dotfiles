Guardar archivos de registro

Como parte de un proceso de solución de problemas, se le podría pedir que guarde los archivos de registro de Junos Pulse. Pulse incluye una cantidad de posibles componentes y cada componente genera uno o más archivos de registro. La operación Guardar como combina todos los archivos de registro de Pulse y otros archivos de diagnóstico en un solo archivo.

Para guardar archivos de registro de Junos Pulse:

  1. En la barra de menú, haga clic en Archivo > Registros > Guardar como. Se mostrará el cuadro de diálogo Guardar como.
  2. Puede aceptar los valores predeterminados para la ubicación y el nombre del archivo o puede especificar nuevos valores y después hacer clic en Guardar.
