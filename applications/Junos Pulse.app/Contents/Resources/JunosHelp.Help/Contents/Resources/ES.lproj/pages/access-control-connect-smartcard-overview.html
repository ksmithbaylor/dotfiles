Autenticación con tarjeta inteligente

La autenticación con tarjeta inteligente utiliza una tarjeta de plástico que incluye información de autenticación del usuario. Si el entorno está configurado con múltiples certificados digitales, Pulse le solicitará que seleccione uno. Por lo general, si dispone de un lector de tarjetas inteligente configurado en su punto final, Pulse interactúa con el software de tarjeta inteligente para recuperar el código token después de que introduzca su número PIN.
