Dodawanie połączenia

Każde połączenie w aplikacji Junos Pulse oznacza sieć chronioną. Zwykle administrator sieci definiuje połączenia wymagane przez użytkownika i może wyłączyć funkcję Add a Connection (Dodaj połączenie). W przypadku konieczności utworzenia nowych połączeń administrator sieci przekaże konieczne ustawienia.

Aby dodać nowe połączenie:

  1. Kliknij przycisk Add a Connection (Dodaj połączenie) w panelu Connections (Połączenia):

    Zostanie otwarte okno dialogowe Add a Connection (Dodaj połączenie).

  2. Administrator sieci wskaże opcję, którą należy wybrać w polu Type (Typ). Wybierz jedno z następujących ustawień:
    • SSL VPN or UAC (SSL VPN lub UAC)—użyj tego typu sieci, jeśli łączysz się z usługą Junos Pulse Secure Access lub Junos Pulse Access Control.
    • SRX—użyj tego typu sieci, jeśli łączysz się z urządzeniem Juniper Networks SRX Series.
  3. W polu Name (Nazwa) wpisz nazwę opisującą połączenie. Wprowadzona nazwa pojawi się w okienku Pulse — Połączenia.
  4. W polu Server URL (Adres URL serwera) określ sieć, z którą chcesz się łączyć. W polu tym można stosować dowolny z następujących formatów:
    • Adres IP, na przykład 10.204.71.86
    • Nazwa DNS, na przykład serwer.mojafirma.net.
  5. Kliknij opcję Add (Dodaj), aby zapisać nowe połączenie i zamknąć okno dialogowe. Kliknij opcję Connect (Połącz), aby zapisać nowe połączenie i zainicjować połączenie z siecią.
