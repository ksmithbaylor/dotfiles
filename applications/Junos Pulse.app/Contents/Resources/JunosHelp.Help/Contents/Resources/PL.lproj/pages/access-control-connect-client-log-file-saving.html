Zapis plików dziennika

Na jednym z etapów rozwiązywania problemu może pojawić się monit o zapisanie plików dziennika aplikacji Junos Pulse. Program Pulse zawiera pewną liczbę składników, z których każdy generuje przynajmniej jeden plik dziennika. Operacja Zapisz jako umożliwia połączenie wszystkich plików dziennika aplikacji oraz innych plików diagnostycznych aplikacji Pulse w jeden plik.

Aby zapisać pliki dziennika programu Junos Pulse:

  1. Na pasku menu kliknij kolejno Plik > Dzienniki > Zapisz jako. Zostanie otwarte okno dialogowe Save As (Zapisz jako).
  2. Zaakceptuj domyślne wartości lokalizacji i nazwy pliku lub wpisz własne, a następnie kliknij przycisk Save (Zapisz).
