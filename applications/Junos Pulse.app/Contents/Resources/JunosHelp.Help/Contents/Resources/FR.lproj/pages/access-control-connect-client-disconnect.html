Interrompre une connexion active

Pour se déconnecter d’un réseau :

Option 1

  1. Dans la fenêtre Pulse, cliquez sur la connexion que vous souhaitez modifier pour la sélectionner.
  2. Dans la barre de menus, cliquez sur Fichier > Connexions > Déconnexion.

Option 2

  1. Dans le volet Connexions de Junos Pulse, cliquez sur Déconnexion pour la connexion à interrompre.

Option 3

  1. Dans la barre de menus, cliquez sur l'icône Pulse.
  2. Cliquez sur la connexion, puis cliquez sur Déconnexion.
