Prolonger une session de connexion

Votre administrateur réseau peut définir les délais d’expiration de session pour vos connexions réseau. Il peut définir une limite maximale du nombre de sessions et une limite d’inactivité de session. Pulse affiche un message lorsque vous atteignez une limite. Le message affiche la durée restante pour la session réseau avant que Pulse interrompe la connexion. En l’absence d’action de votre part, Pulse interrompt la connexion lorsque le compteur de session atteint zéro. Si vous cliquez sur Prolonger, Pulse réinitialise le minuteur sur la valeur maximale définie par l’administrateur réseau et poursuit la session sans interruption de service.

Vous pouvez également prolonger une session sans attendre que Pulse affiche le message d’expiration. Cette opération réalisable à tout moment permet de rétablir les valeurs maximales des minuteurs de session et d’inactivité. Pour prolonger une session :

  1. Dans la liste Connexions de Junos Pulse, cliquez sur la connexion que vous souhaitez prolonger pour la sélectionner.
  2. Dans le menu Pulse, cliquez sur Fichier > Connexions > Prolonger la session.

Ou

  1. Dans la barre de menus, cliquez sur l'icône Pulse pour afficher le menu contextuel.
  2. Cliquez sur la connexion active, puis cliquez sur Prolonger la session.
