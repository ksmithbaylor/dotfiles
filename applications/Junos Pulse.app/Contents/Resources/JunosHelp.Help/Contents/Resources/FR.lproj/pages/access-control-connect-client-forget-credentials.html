Effacer les paramètres enregistrés

Lorsque vous vous connectez à un réseau, vous pouvez cocher la case Enregistrer les paramètres pour que Junos Pulse mémorise vos données d’identification. (Votre administrateur peut désactiver cette fonction.) Chacun des écrans vous invitant à saisir des données comporte sa propre case Enregistrer les paramètres. Lorsque vous enregistrez les paramètres, vous n’êtes plus invité à saisir ces informations lors des prochaines tentatives de connexion. Cependant, si vos données d’identification de connexion changent, vous devrez d’abord effacer les paramètres enregistrés dans Junos Pulse pour pouvoir enregistrer les nouvelles données d’identification. L'option Effacer les paramètres enregistrés concerne tous les paramètres enregistrés et pas uniquement les paramètres de la connexion sélectionnée.

Si vous sélectionnez Effacer les paramètres enregistrés alors qu'une connexion affiche l'état Suspendue, la connexion est interrompue.

L'option de suspension de connexion n'est pas disponible si vous choisissez d'effacer les paramètres enregistrés alors qu'une connexion est active.

Pour effacer les données d’identification enregistrées :

  1. Dans la barre de menus, cliquez sur Fichier > Connexions > Effacer les paramètres enregistrés.
