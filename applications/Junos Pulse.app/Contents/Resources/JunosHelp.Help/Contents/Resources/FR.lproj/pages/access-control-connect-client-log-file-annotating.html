Annoter les fichiers journaux

Lorsque vous annotez un fichier journal, vous insérez du texte marquant le fichier journal à un emplacement spécifique. Par exemple, pour dépanner un problème de connexion, il peut vous être demandé d’annoter le fichier journal avec du texte spécifique, de tenter d’établir la connexion ayant échoué, puis d’annoter de nouveau le fichier journal. Cette séquence d’événements permet à un technicien d’assistance de trouver facilement les entrées du fichier journal assurant le suivi de votre problème de connexion.

Pour annoter les fichiers journaux Junos Pulse :

  1. Dans la barre de menus, cliquez sur Fichier > Journaux > Annoter pour ouvrir la boîte de dialogue Annoter les journaux.
  2. Entrez votre texte d’annotation, puis cliquez sur OK.
