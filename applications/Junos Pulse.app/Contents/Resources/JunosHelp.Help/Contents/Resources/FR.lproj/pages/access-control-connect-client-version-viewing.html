Afficher les informations sur la version du composant

Il est possible que lors d’une opération de dépannage, votre administrateur réseau vous demande de vérifier les numéros de version des programmes de Junos Pulse. Pour accéder à cette information :

  1. Dans la barre de menus, cliquez sur Junos Pulse > À propos de Junos Pulse pour ouvrir la boîte de dialogue À propos.
  2. Cliquez sur le numéro de la version pour ouvrir la boîte de dialogue Détails de la version de Pulse.
