Authentification par jeton

Pulse prend en charge le système d’authentification par jeton RSA SecurID. L’authentification par jeton, également appelée authentification à deux facteurs, est une méthode consistant à générer des données d’identification extrêmement difficiles à compromettre. Un code de jeton est un numéro généré en même temps sur le client et sur le serveur. L’utilisateur crée un numéro d’identification personnel (code PIN). Le nom d’utilisateur et le code PIN combinés au code de jeton permettent ensuite à l’utilisateur de se connecter au serveur. Un code PIN et un code de jeton combinés forment un mot de passe. Un code de jeton expire après un court intervalle, généralement d’une minute, puis il est remplacé par un nouveau code de jeton généré.

La première fois que vous utilisez le système, Pulse vous invite à créer un code PIN. Si votre administrateur a déjà fourni un code PIN, vous êtes invité à le modifier. Si votre administrateur de réseau exige que vous changiez votre code PIN régulièrement, Pulse vous invite à entrer le nouveau code PIN. En principe, vous devez fournir le code de jeton actuel lorsque vous modifiez votre code PIN. Dans certains cas, après avoir entré votre mot de passe, vous êtes invité à entrer le prochain code de jeton pour pouvoir accéder au réseau. Attendez juste que le dispositif de jeton change le code de jeton puis entrez ce numéro pour continuer.

Authentificateurs

Le dispositif ou logiciel qui génère le jeton est appelé un authentificateur. Pulse prend en charge une gamme complète d’authentificateurs matériels SecurID.

Les authentificateurs matériels (physiques) RSA SecurID affichent un nouveau code de jeton toutes les 60 secondes. Vous devez disposer de la carte ou du porte-clés qui génère le jeton avec vous lorsque vous vous connectez. Dans la fenêtre de connexion Pulse, fournissez votre nom d’utilisateur et votre code PIN avec le code de jeton actuel. Un authentificateur USB peut mémoriser vos nom d’utilisateur/mot de passe Mac OS et vos certificats numériques. Lorsque le dispositif est connecté à votre ordinateur, il permet l’entrée automatique du code de jeton. Pulse peut alors accéder automatiquement aux codes de jeton dès leur sortie du dispositif une fois que vous avez entré votre code PIN.
