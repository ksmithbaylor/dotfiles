Afficher les détails de connexion avancés

Le volet Connexions contient des informations de base sur une connexion. Vous pouvez également consulter les détails de connexion avancés pour visualiser des informations détaillées sur une connexion donnée.

Pour afficher les détails de connexion avancés :

Option 1

  1. Cliquez sur la connexion pour la sélectionner.
  2. Dans la barre de menus, cliquez sur Fichier > Connexions > Détails de connexion avancés.

Option 2

  1. Cliquez avec le bouton droit de la souris sur la connexion Pulse pour afficher le menu contextuel.
  2. Cliquez sur Détails de connexion avancés.
