Fichiers journaux Junos Pulse

Les fichiers journaux Junos Pulse établissent le suivi des informations permettant de résoudre les problèmes de connexion. La journalisation est une opération d’arrière-plan. Il n’est pas nécessaire d’apporter des modifications à votre environnement de journalisation, sauf si vous y êtes invité dans le cadre du dépannage.

Les fichiers journaux comprennent vos connexions Pulse et leurs propriétés. Si vous souhaitez enregistrer la configuration, il suffit d’enregistrer tous les fichiers journaux. Votre configuration Pulse actuelle est enregistrée dans le module des journaux. Si vous désinstallez Pulse, le programme vous demande si vous souhaitez enregistrer votre configuration Pulse ; ainsi, lorsque vous réinstallerez Pulse ultérieurement, vous récupérerez vos connexions Pulse et les autres paramètres de configuration.

Pour faciliter le dépannage, il peut vous être demandé d’effectuer les tâches décrites ci-dessous.

  • Annoter les journaux : lorsque vous annotez un fichier journal, vous insérez du texte marquant le fichier journal à un emplacement spécifique. Par exemple, pour dépanner un problème de connexion, il peut vous être demandé d’annoter le fichier journal avec du texte spécifique, de tenter d’établir la connexion ayant échoué, puis d’annoter de nouveau le fichier journal. Cette séquence d’événements permet à un représentant de l’assistance de rechercher le texte que vous avez inséré dans le fichier journal. Le texte met entre crochets les entrées du fichier journal établissant le suivi de votre problème de connexion.
  • Définir le niveau de journalisation : le niveau de journalisation par défaut est Normal. Pour une opération de dépannage, il peut vous être demandé de définir le niveau de journalisation sur Détaillé.
  • Enregistrer les journaux : l’opération Enregistrer sous rassemble tous les fichiers journaux dans un fichier unique et vous permet de spécifier où le placer.
