Ajouter une connexion

Dans Junos Pulse, chaque connexion représente un réseau protégé. L’administrateur réseau définit généralement vos connexions et peut désactiver la fonction Ajouter une connexion. Si vous devez créer des connexions, l’administrateur réseau vous dira quels paramètres utiliser.

Pour ajouter une nouvelle connexion :

  1. Dans le volet Connexions, cliquez sur le bouton Ajouter une connexion :

    La boîte de dialogue Ajouter une connexion s’ouvre.

  2. Pour le type, l’administrateur réseau vous indiquera l’option à sélectionner. Choisissez l’une des options suivantes :
    • SSL VPN or UAC—utilisez ce type de réseau si vous vous connectez à Junos Pulse Secure Access Service ou à Junos Pulse Access Control Service.
    • SRX—utilisez ce type de réseau si vous vous connectez à un dispositif SRX Juniper Networks.
  3. Pour le nom, entrez un nom descriptif pour cette connexion. Le nom que vous spécifiez apparaît dans le volet Connexions de Junos Pulse.
  4. Pour l’adresse URL du serveur, spécifiez le réseau auquel vous voulez vous connecter. Vous pouvez entrer l’adresse URL du serveur dans l’un des formats suivants :
    • Une adresse IP, par exemple 10.204.71.86
    • Un nom DNS, par exemple serveur.masociete.net.
  5. Cliquez sur Ajouter pour enregistrer votre nouvelle connexion et fermer la boîte de dialogue. Cliquez sur Connexion pour enregistrer votre nouvelle connexion et établir une connexion au réseau.
