要求技術支援

技術產品支援可透過「Juniper Networks 技術支援中心」(JTAC) 取得。 如果您是有效 J-Care 或 JNASC 支援合約或有效保固期限之客戶，且需要售後技術支援，您可以在線上存取我們的工具及資源，或透過 JTAC 建立案例。

  • 如需完全瞭解我們的 JTAC 程序與政策，請參閱位於下列位址的《JTAC 使用者指南》
    http://www.juniper.net/customers/support/downloads/7100059-EN.pdf
  • 如需產品保固資訊，請造訪
    http://www.juniper.net/support/warranty/

JTAC 中心 24 小時全天候提供服務，全年無休。

線上自助說明工具與資源

為了快速輕鬆地解決問題，Juniper Networks 已設計出一種稱為「客戶支援中心」(CSC) 的線上自助服務入口，可為您提供下列功能：

  • 尋找 CSC 提供的內容，網址為：
    http://www.juniper.net/customers/support/
  • 尋找產品文件，網址為：
    http://www.juniper.net/techpubs/
  • 使用我們的「知識庫」尋找解決方案並回答問題，網址為：
    http://kb.juniper.net/
  • 下載最新版本的軟體及檢視版本說明，網址為：
    http://www.juniper.net/customers/csc/software/
  • 在技術布告欄中搜尋相關硬體與軟體通知，網址為：
    http://www.juniper.net/alerts/
  • 加入並參與 J-Net 論壇，網址為：
    http://www.juniper.net/company/communities/
  • 在 CSC Case Manager 中線上建立案例，網址為：
    http://www.juniper.net/customers/cm/
  • 使用我們的「序號權利」(SNE) 工具，依據產品序號確認服務權利，網址為：
    https://tools.juniper.net/SerialNumberEntitlementSearch/

使用 JTAC 建立案例

您可以透過 Web 或電話向 JTAC 建立案例。

  • 使用 CSC 中的 Case Manager 工具，網址為：
    http://www.juniper.net/customers/cm/
    致電 1-888-314-JTAC (美國、加拿大及墨西哥境內免付費電話：1-888-314-5822)。
  • 如需未提供免付費電話之國家/地區的國際或直撥電話，請造訪：
    http://www.juniper.net/support/requesting-support.html
