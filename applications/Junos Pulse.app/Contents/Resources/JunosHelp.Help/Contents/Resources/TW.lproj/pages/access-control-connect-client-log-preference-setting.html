設定日誌層級

若要設定日誌層級：

  1. 在功能表列上，按一下 File [檔案] > Logs [日誌] > Log Level [日誌層級] > Detailed [詳細] 或 File [檔案] > Logs [日誌] > Log Level [日誌層級] > Normal [正常]。 核取標記代表該選項目前已啟用。

    2. 預設值是 Normal [正常] 記錄。 除非要對連線問題進行疑難排解，否則應該將記錄設定為 Normal [正常]。

    Detailed [詳細] 記錄會建立更大量的日誌項目，並會增加日誌檔的大小。 Detailed [詳細] 記錄通常只在對問題進行疑難排解時才啟用。
