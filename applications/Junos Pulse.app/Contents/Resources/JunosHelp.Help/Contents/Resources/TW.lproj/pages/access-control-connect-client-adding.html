新增連線

Junos Pulse 中的每個連線都代表一個受保護的網路。 一般而言，您的網路管理員會為您定義連線，且可能會停用 Add a Connection [新增連線] 功能。 如果您需要建立新連線，您的網路管理員會告知您必須使用的設定。

若要新增連線：

  1. 在 Connections [連線] 窗格上，按一下 Add a Connection [新增連線] 按鈕：

    Add Connection [新增連線] 對話方塊隨即開啟。

  2. 針對 Type [類型]，您的網路管理員通常會告知您應選取的選項。 請選擇下列其中一項：
    • SSL VPN or UAC—如果您是連線至 Junos Pulse Secure Access Service 或 Junos Pulse Access Control Service，請使用此網路類型。
    • SRX—如果您是連線至 Juniper Networks SRX 系列裝置，請使用此網路類型。
  3. 針對 Name [名稱]，請為此連線指定一個說明性的名稱。 您指定的名稱會顯示在 Pulse 的 Connections [連線] 窗格中。
  4. 針對 Server URL [伺服器 URL]，請指定您要連線的目標網路。 您可以使用下列其中任一種格式輸入伺服器 URL：
    • IP 位址，例如 10.204.71.86
    • DNS 名稱，例如 server.mycompany.net。
  5. 按一下 Add [新增]，儲存您的新連線並關閉對話方塊。 按一下 Connect [連線]，儲存您的新連線並初始化與網路的連線。
