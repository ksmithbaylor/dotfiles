儲存日誌檔

作為疑難排解程序的一部分，系統可能會要求您儲存 Junos Pulse 日誌檔。 Pulse 會加入許多可能的相關元件，且每個元件都會產生一或多個日誌檔。 Save As [另存新檔] 操作會將所有 Pulse 日誌檔與其他診斷檔案結合成一個檔案。

若要儲存 Junos Pulse 日誌檔：

  1. 在功能表列上，按一下 File [檔案] > Logs [日誌] > Save As [另存新檔]。 Save As [另存新檔] 對話方塊隨即出現。
  2. 接受位置與檔案名稱的預設值，或者指定新值，然後按一下 Save [儲存]。
