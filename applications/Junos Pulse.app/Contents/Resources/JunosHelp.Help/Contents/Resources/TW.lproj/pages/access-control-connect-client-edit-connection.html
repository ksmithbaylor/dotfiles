編輯連線屬性

建立連線後，您可以編輯 URL 及顯示在 Connections [連線] 窗格中的名稱。 只有在連線目前並未處於作用中的狀態時，您才能編輯該連線。

若要編輯連線：

選項 1

  1. 在 Pulse 視窗中，按一下以選取要編輯的連線。
  2. 在功能表列上，按一下 File [檔案] > Connections [連線] > Edit [編輯]。

    Edit Connection [編輯連線] 對話方塊隨即開啟。

  3. 編輯連線，然後按一下 Save [儲存] 以儲存變更並關閉對話方塊，或者按一下 Connect [連線] 以初始化連線並關閉對話方塊。

選項 2

  1. 按一下該連線加以選取。
  2. 按一下 Edit Connection [編輯連線] 按鈕：

    Edit Connection [編輯連線] 對話方塊隨即開啟。

  3. 編輯連線，然後按一下 Save [儲存] 以儲存變更並關閉對話方塊，或者按一下 Connect [連線] 以初始化連線並關閉對話方塊。
