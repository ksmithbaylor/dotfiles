For modifications in this files, please use Text Editors which persist these BOM bytes (like Notepad.exe, Visual Studio IDE, etc).

;kMsgCommonCertTrustNoError
[1700]
Short-desc = No se encontraron errores para este certificado o cadena.
Long-desc = El recurso de red para esta conexión se ha verificado y es confiable.

;kMsgCommonCertTrustIsNotTimeValid
[1701]
Short-desc = Este certificado o uno de los certificados en la cadena de certificado no tiene una hora válida.
Long-desc = Es posible que la hora del sistema en su punto final sea incorrecta, el certificado puede haber caducado o la hora del sistema que firmó el certificado puede haber sido incorrecta. Verifique la precisión del reloj del equipo. Si el reloj del equipo no es preciso, entonces no se puede verificar el recurso de red para esta conexión. Comuníquese con el administrador de la red.
