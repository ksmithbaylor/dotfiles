; JUNS catalogUtil requires message catalog files to be saved with UTF-8 encoding, which put 3 bytes {0xEF, 0xBB, 0xBF} at the file beginning called Byte Order Mark (BOM) for UTF-8 files . For modifications in this files, please use Text Editors which persist these BOM bytes (like Notepad.exe, Visual Studio IDE, etc).

;kMsgCommonCertTrustNoError
[1700]
Short-desc = Nessun errore trovato per questo certificato o catena.
Long-desc = La risorsa di rete per la connessione è stata verificata e può essere ritenuta attendibile.

;kMsgCommonCertTrustIsNotTimeValid
[1701]
Short-desc = Il certificato o uno dei certificati nella catena non è compreso nel periodo di validità.
Long-desc = L'ora di sistema nell'endpoint potrebbe essere errata, il certificato potrebbe essere scaduto oppure l'ora del sistema che ha firmato il certificato potrebbe essere errata. Verificare che il clock nel computer indichi l'ora precisa. Se il clock nel computer è preciso, non è possibile verificare la risorsa di rete per questa connessione. Contattare l'amministratore della rete.
