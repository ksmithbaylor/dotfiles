# Example skill

A starter skill. To author your own:

1. Copy this folder to `skills/<your-skill-name>/` (kebab-case).
2. Set `name` (must match the folder) and a trigger-focused `description`.
3. Write the instructions the agent should follow here, replacing this body.
4. Run `./install` (or just the skills-symlink loop) so it links into both
   `~/.claude/skills` and `~/.codex/skills`.
