
This is the J802 distribution.

For more information see: http://www.jsoftware.com/jwiki/System/Installation.

Shortcuts:
jbrk - send break signal to J
jcon - load Jconsole
jhs  - start JHS server
jqt  - load JQt (if the Qt IDE is installed)

Shell scripts:
updatejqt.sh       - update JQt binaries (if the Qt IDE is installed)
